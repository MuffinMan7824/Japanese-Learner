This is a basic app to help you learn Japanese this is all very early beta

So far all you can do is click on a letter and the get the sound but later more features will be added 

Installation 

1. Clone the repository 

   git clone https://github.com/MuffinMan7824/Japanese-Learner.git
   
   cd Japanese-Learner

2. Create a virtual environment

   python -m venv .venv

3. Activate the virtual environment

   Windows:

   .venv\Scripts\activate

   Mac/Linux:

   source .venv/bin/activate

4. Install dependencies

   pip install -r requirements.txt

5. Run the app

   python main.py
